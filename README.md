# EcoLifestyleCoach

AI-powered eco-friendly lifestyle coach using GPT-OSS model.

## Getting Started

1. Clone the repository: `git clone https://github.com/your-username/EcoLifestyleCoach.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`

## Testing

1. Use the provided sample data in `data/sample_input.json`
2. Run the application with the sample data: `python app.py --input data/sample_input.json`

## GPT-OSS Model Usage

We fine-tuned the GPT-OSS model on our dataset to generate personalized eco-friendly recommendations. The model weights are available in `models/fine_tuned_model.pth`.

## Example Use Cases

* Get personalized recommendations for reducing energy consumption
* Receive suggestions for eco-friendly transportation options
